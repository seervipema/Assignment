# Instructions to run the application
 - Install Node version v18.10.0
 - Step 1 : nvm install v18.10.0
 - Step 2 : nvm use v18.10.0
 - Step 2 : npm install
 - Step 3 : npm test
 - Step 4 : npm start
 - Step 5 : Open browser and hit this URL http://localhost:3000/
